Algoritmo de Validação do CPF

Como os dígitos verificadores são calculados — do zero, passo a passo

Teste um CPF

Como o algoritmo funciona

O CPF tem 11 dígitos. Os 9 primeiros identificam o contribuinte; os 2 últimos são os dígitos verificadores — calculados a partir dos anteriores. Para validar um CPF, recalculamos esses dois dígitos e conferimos se batem com os informados.

Usaremos o CPF fictício 111.444.777-35 como exemplo.

Passo 1 — Calcular o primeiro dígito verificador

Pegamos os 9 primeiros dígitos (111.444.777) e multiplicamos cada um por um peso decrescente, de 10 a 2, da esquerda para a direita:

Dígito 1 1 1 4 4 4 7 7 7
Peso 10 9 8 7 6 5 4 3 2
Produto 10 9 8 28 24 20 28 21 14

Somamos todos os produtos:

10 + 9 + 8 + 28 + 24 + 20 + 28 + 21 + 14 = 162

Dividimos por 11 e ficamos com o resto:

162 ÷ 11 = 14, resto 8

— Se o resto for menor que 2: o dígito verificador é 0

— Se o resto for maior ou igual a 2: o dígito verificador é 11 − resto

Resto 8 ≥ 2, então: 11 − 8 = 3 → primeiro dígito verificador = 3

CPF parcial: 111.444.777-3X

Passo 2 — Calcular o segundo dígito verificador

Agora usamos os 9 dígitos originais mais o primeiro verificador (10 dígitos no total) e os multiplicamos por pesos de 11 a 2:

Dígito 1 1 1 4 4 4 7 7 7 3
Peso 11 10 9 8 7 6 5 4 3 2
Produto 11 10 9 32 28 24 35 28 21 6

Somamos os produtos:

11 + 10 + 9 + 32 + 28 + 24 + 35 + 28 + 21 + 6 = 204

Dividimos por 11 e ficamos com o resto:

204 ÷ 11 = 18, resto 6

Resto 6 ≥ 2, então: 11 − 6 = 5 → segundo dígito verificador = 5

CPF completo confirmado: 111.444.777-35 ✓

Casos especiais

CPFs formados por dígitos iguais — como 000.000.000-00 ou 111.111.111-11 — passariam na conta matemática, mas são inválidos por definição. O algoritmo deve rejeitá-los explicitamente.

Implementação em JavaScript

validarCPF.js 
/**
 * Valida um CPF brasileiro.
 * Aceita formatos: "111.444.777-35" ou "11144477735"
 * Retorna true se o CPF é válido, false caso contrário.
 */
function validarCPF(cpf) {

  // Remove tudo que não for dígito
  const numeros = cpf.replace(/\D/g, '');

  // Deve ter exatamente 11 dígitos
  if (numeros.length !== 11) return false;

  // CPFs com todos os dígitos iguais são inválidos (ex.: 111.111.111-11)
  if (/^(\d)\1{10}$/
.test(numeros)) return false;

  // Calcula um dígito verificador a partir de um subconjunto do CPF
  function calcularDigito(base, pesoInicial) {
    const soma = base
      .split('')
      .reduce((acumulador, digito, indice) => {
        return acumulador + Number(digito) * (pesoInicial - indice);
      }, 0);

    const resto = soma % 11;
    return resto < 2 ? 0 : 11 - resto;
  }

  // Verifica o primeiro dígito (usa os 9 primeiros, pesos de 10 a 2)
  const digito1 = calcularDigito(numeros.slice(0, 9), 10);
  if (digito1 !== Number(numeros[9])) return false;

  // Verifica o segundo dígito (usa os 10 primeiros, pesos de 11 a 2)
  const digito2 = calcularDigito(numeros.slice(0, 10), 11);
  if (digito2 !== Number(numeros[10])) return false;

  return true;
}
